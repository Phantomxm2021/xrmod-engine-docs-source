## Description

BaseNotificationData的子类。当使用图像跟踪时，ARMOD SDK事件接收器功能将接收这种类型的数据。


## Parameters

| Param         | Type          | Description                                                    |
| ------------- | ------------- | -------------------------------------------------------------- |
| MapId         | int           | Current map id                                                 |
| LocalizerPose | LocalizerPose | You should set trackabletype when stick type is ByTackableTYpe |
