## What is Remote debug

一个用于 AR 体验开发的远程调试工具。它可以帮助我们更快地迭代我们的 AR 体验。

目前，由于我们的人力不足，我们暂时不会提供 AR-MOD 远程调试，但不要失望。我们已经从网上找到了几个解决方案，供大家使用和参考!
