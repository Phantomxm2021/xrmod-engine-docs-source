title: Programmable Block
---

额外的脚本输入字段，用于配置当前的 AR 体验。相应的程序已经默认匹配，不需要手动配置。

## Programmable Type

:::caution
C#脚本不能与可视化脚本混合使用。
:::

正如它的名字一样。这个选项允许你选择编写 C#脚本或 Visual 脚本。

## Domain Name

与代码相对应的命名空间；如果配置不正确，代码将无法执行。

## Main Entry

代码的具体条目；如配置错误，代码不能被执行。

## Debug-Model

打开伪调试模式（可打印信息），但打开这个选项会影响 AR 体验执行的性能。请谨慎使用!
