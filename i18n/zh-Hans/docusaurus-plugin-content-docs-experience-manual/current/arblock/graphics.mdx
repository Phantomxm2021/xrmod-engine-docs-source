# Quality

我们知道，质量设置在 unity 中是非常重要的。适当的设置会使我们渲染的图像看起来更好，并以最佳性能运行。Unity 提供了 6 个默认的质量级别供我们选择。我们可以在 Property 中设置它。我们可以添加质量块来在运行时设置我们的图形级别。

> Unity 允许你设置它试图渲染的图形质量水平。一般来说，质量是以牺牲帧率为代价的，所以在移动设备或旧硬件上最好不要追求最高质量，因为这往往会对游戏性产生不利影响。使用质量设置（菜单："编辑">"项目设置"，然后选择 "质量 "类别），在编辑器中为所选设备选择质量级别。它分为两个主要区域：质量矩阵出现在顶部；在它下面，出现所选质量级别的设置。
> From:[Unity Docs](https://docs.unity3d.com/2020.3/Documentation/Manual/class-QualitySettings.html)

:::caution
如果你在编辑器中设置了质量等级，那么它只能在编辑器中使用，而不能在 XRMOD SDK 中使用。XRMOD 默认运行在低级别。如果你的着色器使用 Stencil, Depth Texture ,HDR 等，请将其设置为高等级。
:::
