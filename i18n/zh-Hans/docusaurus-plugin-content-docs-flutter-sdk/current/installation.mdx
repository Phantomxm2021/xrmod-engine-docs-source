XRMOD 引擎提供了一个在跨平台上实现增强现实功能的解决方案。而且你不需要知道如何用原生代码构建 3D 内容，当你的应用需要添加新的 AR 体验时，你也不需要重新向应用商店提交。XRMOD 引擎有许多 AR 功能，如寻找平面放置虚拟物体、追踪图像、追踪人脸或多人 ar 游戏。

XRMOD 引擎是一个免费的、无水印的元空间构建系统，支持商业许可（免费商业许可）。

在跨平台上嵌入 XRMOD 引擎不需要改变你原有的业务和代码，只需要调用几个 API 即可完成嵌入。
