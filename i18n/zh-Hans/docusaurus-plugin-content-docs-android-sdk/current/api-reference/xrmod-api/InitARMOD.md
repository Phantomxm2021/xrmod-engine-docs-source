## Description

使用配置信息初始化 ARMOD SDK。此方法将自动调用。

## Parameters

| Name         | Type     | Description                                                   |
| ------------ | -------- | ------------------------------------------------------------- |
| appconfigure | NSString | 这是一个 json 字符串。用来启动我们的 SDK 和认证。 |
| Class        | Activity | 关闭 ARMOD 后显示的视图。                            |

## Method

```java
public void initARMOD(String _appConfigure, Class<?> _activity)
```
