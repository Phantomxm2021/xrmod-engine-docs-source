## Description

获取当前的 ARMOD FrameLayout。你可以调用此方法来获得框架布局，并增加你的自定义 UI。


## Return

| Type                                               | Description                 |
| -------------------------------------------------- | --------------------------- |
| <highlight color="#BA89F3">framelayout</highlight> | current ARMOD's framelayout |

## Method
```java
public FrameLayout getARMODFrameLayout()
```
