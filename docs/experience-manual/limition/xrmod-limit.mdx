title: XRMOD Limited
---

- Turning off will still occupy 100-200M of running memory (including the memory allocated by ARKit)
- The CPU occupancy rate depends on the content, under normal circumstances the occupancy rate is 70%
- Dose not support loading multiple ARMOD runtime instances
- Only full-screen rendering is supported, and rendering on a part of the screen is not supoorted
