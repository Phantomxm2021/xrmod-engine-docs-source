## Description

从注册列表中获取网络预制件。

## Return

| Type       | Description        |
| ---------- | ------------------ |
| GameObject | Network GameObject |

## Parameters

| Connection  | Type   | Description                                      |
| ----------- | ------ | ------------------------------------------------ |
| Prefab Name | String | The name of game object in the registration list |
