## Return

| Type | Description    |
| ---- | -------------- |
| Bool | True or False. |

## Parameters

| Connection | Type            | Description                                    |
| ---------- | --------------- | ---------------------------------------------- |
| Target     | NetworkIdentity | Need to check of authority network game object |
