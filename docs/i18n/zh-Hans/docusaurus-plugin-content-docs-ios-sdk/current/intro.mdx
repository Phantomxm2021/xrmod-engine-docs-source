> <h3>潜入增强现实的世界中</h3>
>
> 在全球最大的AR平台iOS和iPadOS上为数以亿计的用户打造无与伦比的增强现实体验。有了像ARKit和RealityKit这样强大的框架，以及像Reality Composer和Reality Converter这样的创意工具，将你的想法变成现实的AR从未如此简单。 
>
> from https://developer.apple.com/augmented-reality/

XRMOD引擎为在iOS平台上实现增强现实功能提供了一个解决方案。而且你不需要知道如何用原生代码构建3D内容，当你的应用需要添加新的AR体验时，也不需要重新向应用商店提交。XRMOD引擎有许多AR功能，如寻找平面放置虚拟物体、追踪图像、追踪人脸或多人ar游戏。

XRMOD引擎是一个免费的、无水印的元空间构建系统，支持商业许可（免费商业许可）。


在iOS平台上嵌入XRMOD引擎不需要改变你原有的业务和代码，只需要调用几个API即可完成嵌入。
