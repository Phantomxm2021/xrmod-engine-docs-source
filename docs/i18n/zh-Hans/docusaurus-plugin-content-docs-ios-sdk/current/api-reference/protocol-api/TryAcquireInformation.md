## Description

从 ARExperience 脚本中获取应用信息

## Parameters

| Param    | Description                                               |
| -------- | --------------------------------------------------------- |
| opTag    | 由 AR Experience 发起的操作请求的类型。 |
| callback | 执行回调（异步）。                          |

## Method 

```objectivec
- (void )tryAcquireInformation:(NSString*) opTag CallBackFuncP:(TryAcquireInformationCallBackFuncP) callback;
```
