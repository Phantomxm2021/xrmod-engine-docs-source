## Description

检测到 ARExperience 软件包太大

## Parameters

| Param       | Description           |
| ----------- | --------------------- |
| currentSize | 目前的包大小(M)       |
| presetSize  | 最大的可下载包大小(M) |

## Method

```objectivec
- (void )packageSizeMoreThanPresetSize:(float) currentSize preset:(float) presetSize;
```
