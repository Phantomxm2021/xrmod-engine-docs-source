## Description

设置 AR 视图以跟随应用程序本身的方向

## Parameters

| Name        | Type                   | Description               |
| ----------- | ---------------------- | ------------------------- |
| orientation | UIInterfaceOrientation | 当前应用程序的方向 |

## Method 

```objectivec
- (void)setUIInterfaceOrientation:(UIInterfaceOrientation)orientation
```
