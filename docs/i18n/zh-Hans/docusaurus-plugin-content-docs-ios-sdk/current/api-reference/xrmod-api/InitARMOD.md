## 描述

使用配置信息初始化 ARMOD SDK。此方法将自动调用。

## 参数

| Name         | Type     | Description                                                   |
| ------------ | -------- | ------------------------------------------------------------- |
| appconfigure | NSString | 这是一个 json 字符串。用来启动我们的 SDK 和认证 |

## 方法

```objectivec
- (void)initARMOD:(NSString*) appconfigure
```
