## 描述

仪表板的配置

| Name                | Type   | 描述                |
| ------------------- | ------ | ------------------- |
| dashboardGateway    | string | 服务网关（API URI） |
| token               | string | 帐户认证令牌      |
| timeout             | int    | 最长超时时间        |
| maximumDownloadSize | float  | 最大文件下载大小    |
