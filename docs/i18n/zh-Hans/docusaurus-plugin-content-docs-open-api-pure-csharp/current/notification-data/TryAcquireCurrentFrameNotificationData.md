
## 描述

BaseNotificationData 的子类。当您调用 TryAcquireCurrentFrame 方法时，将需要将此数据传递给 ARMODSDK。

## 参数

| Param                 | Type           | 描述               |
| --------------------- | -------------- | ------------------------- |
| AcquiredTextureFormat | TextureFormat  | 纹理格式            |
| ConversionType        | ConversionType | 同步的，异步的 |
