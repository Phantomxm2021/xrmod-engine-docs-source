
## Description


BaseNotificationData 的子类。使用图像跟踪时，这种类型的数据。


## Parameters

| Param      | Type        | Description             |
| ---------- | ----------- | ----------------------- |
| FocusPos   | Vector3     | Focus's position        |
| FocusRot   | Vector3     | Focus's rotation        |
| FocusState | FindingType | Current tracking status |
