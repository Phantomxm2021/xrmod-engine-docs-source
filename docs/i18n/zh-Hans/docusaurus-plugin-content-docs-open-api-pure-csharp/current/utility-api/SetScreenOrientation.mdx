:::caution
逻辑方向不仅影响屏幕方向，而且也影响触摸坐标。你应该期待改变逻辑方向后触摸位置的剧烈变化，因为触摸位置会顺时针或逆时针旋转以匹配屏幕坐标。
:::

## 描述

规定屏幕的逻辑方向。

## Parameters

| Param         | Type              | 描述     |
| ------------- | ----------------- | -------- |
| \_orientation | ScreenOrientation | 屏幕方向 |

:::tip Screen Orientation

| Id  | Orientation        | 描述                               |
| --- | ------------------ | ---------------------------------- |
| 0   | Portrait           | 纵向                               |
| 1   | PortraitUpsideDown | 纵向，颠倒的方向                   |
| 2   | LandscapeLeft      | 横向方向，从纵向方向逆时针旋转     |
| 3   | LandscapeRight     | 横向方向，从纵向方向顺时针方向排列 |

:::

## 方法

```cs
public void SetScreenOrientation(ScreenOrientation _orientation)
```

## 例子

```cs
internal static API ARMODAPI = new API(nameof(APITest));
private void OnLoad(){
    ChangeScreenOrientation();
}
private void ChangeScreenOrientation(){
    ARMODAPI.SetScreenOrientation(ScreenOrientation.Portrait);
}
```
