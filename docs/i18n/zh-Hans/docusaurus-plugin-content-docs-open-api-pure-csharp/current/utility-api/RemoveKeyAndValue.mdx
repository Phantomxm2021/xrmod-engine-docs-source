## 描述

本地存储临时数据。`SaveKeyAndValue`是一个在项目会话之间存储经验案例偏好的方法。它可以将字符串存储到用户的平台注册表中。

从 lcaolly 中删除给定的键。如果该键不存在，`RemoveKeyAndValue`没有影响。

## 参数

| Param         | 描述                                  |
| ------------- | ------------------------------------- |
| \_projectName | 使用项目名称+密钥存储，防止密钥被占用 |
| \_key         | 独一无二的名称                        |

## 方法

```cs
public void RemoveKeyAndValue(string _projectName,string _key)
```

## 例子

```cs
internal static API ARMODAPI = new API(nameof(APITest));
private void OnLoad(){
    ARMODAPI.RemoveKeyAndValue(nameof(APITest),"StoreKey");
}
```
