
## Description

比较两个标签是否一致。它类似于[Unity的CompareTag](https://docs.unity3d.com/ScriptReference/Component.CompareTag.html)方法。

## 参数

| Param | Type   | Description |
| ----- | ------ | ----------- |
| _tag | String | 独一无二的名字 |

## 方法

```cs
public void DoCompareTag(string _tag)
```

## 例子

::: Caution
你必须将`GameObjectRuntimeTag`脚本添加到你将用于检索的gameObject中，并在`GameObjectRuntimeTag.RuntimeTag`字段中输入`Tag`。
:::

```cs
private void OnLoad(){
    RuntimeTagManager.GetRuntimeTagManager.GetGameObjectByTag("Player",out GameObject tmp_Player);
    if(tmp_Player.GetComponent<IRuntimeTag>().DoCompareTag("Player")){
        Debug.Log(tmp_Player.name);
    }
}
```
