## Parameters

| Connection  | Description                                    |
| ----------- | ---------------------------------------------- |
| Server Type | What kind of server will be created.           |
| ServerType  | Description                                    |
| Host        | Create a server and client on the same device. |
| Server Only | Only createt a server.                         |
